This repository contains input xml files for Cyclus simulation which simulates the following benchmark:

B. Feng, et al., Standardized verification of fuel cycle modeling, Annals of Nuclear Energy Volume 94, August 2016, Pages 300-312

The original description for the input files (the originality) can be found below:
https://github.com/arfc/transition-scenarios/tree/master

It was included in Jin Hwan Bae's Master's Thesis. (For clairy, both the original papers are also uploaded in this repository)

Run the input xml files with the following output names:
ㅁ output_Original
ㅁ output_TsOh

Then, run the python scripts to visualize the results.
All files & folder(s) should be downloaded to run properly.
